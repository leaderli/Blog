jenkins远程发布

发表于 2019-07-22 分类于 jenkins

版本说明

省略docker安装过程

docker version

java -version

安装Jenkins镜像

拉取的是2.138.1-slim版本的jenkins，可通过命令查看下载的版本

docker images

启动Jenkins

8002:8080 本地8002端口映射容器8080端口

查看启动日志 docker logs -f jenkins 可以看到jenkins的默认admin密码

界面访问127.0.0.1:8002,自动跳转至登录界面,首次进线输入默认密码，登录后安装推荐插件。

jenkins插件镜像

页面依次点击/Manage Jenkins/Manage Plugins/Advanced 镜像地址：

# docker # jenkins # tomact
